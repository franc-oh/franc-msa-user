# SpringCloud MSA - User Service
> 스프링클라우드를 통한 MSA 구축 - 회원 서비스

<br/>

## ⚙️ 개발 환경
- JDK 17
- Spring Boot 3.2.1
- Spring Cloud Eureka Client 4.1.3
- Spring Date JPA
- Lombok
- H2 DB
- Gradle 8.10

<br/>
